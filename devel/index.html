Getting involved

We always need developers, tech writers and testers to work on making nms better. Please visit the nms project page on Sourceforge for more details on getting involved.

Currently our biggest requirement is for people to get involved with the support mailing list. If you'd like to help out answering people's questions about our programs then please let us know.

There are currently two mailing lists set up for the discussion of nms.

  • The nms end-user support list
  • The nms developers discussion list

Future Plans

The initial plan is to create a set of scripts that mirror exactly the functionality of Matt's scripts (minus the insecurities and bugs). This is now complete. Now I'd like to look at extending the archive in four directions:

  • Simplifying and improving the installation process.
  • Adding new functionality to the existing scripts.
  • Adding new scripts that are not in the original list.
  • Adding enhanced versions of the scripts which require slightly more knowledge from the person installing them - perhaps needing modules from CPAN.

Any suggestions in any of these areas will be appreciated.
