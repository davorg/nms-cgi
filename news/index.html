nms in Debian Stable

The "testing" release of the Debian Linux distribution which includes nms packages has been promoted to Debian stable.

Just as soon as the Debian web server wakes up a bit, I'll update this item with a list of the package names.

nms goodies at Cafepress

You can now buy a range of nms stuff at Cafepress. Any profits from this store will be donated to The Perl Foundation.

nms as recommended by Matt Wright

Matt Wright has given the MSA web site a major redesign. He's included a page about nms. Here are a couple of extracts.

While the free code found at my web site has not evolved much in recent years, the general programming practices and standards of CGI programs have. nms is an attempt by very active programmers in the Perl community to bring the quality of code for these types of programs up to date and eliminate some of the bad programming practices and bugs found in the existing Matt's Script Archive code.
I would highly recommend downloading the nms versions if you wish to learn CGI programming. The code you find at Matt's Script Archive is not representative of how even I would code these days.

I call that a good result :)

MSA FormMail Attacks

Security Focus put MSA FormMail attacks third in their list of the Top Attacks for the 1st Quarter of 2002.

nms included in Debian Linux

Debian is one of the best-known and well-respected distributions of Linux. Thanks to the efforts of Christoph Martin, prepacked versions of many nms programs are amongst the packages included in their current testing release. This is the version that will be released as the next stable release in due course.

Many thanks to Christoph for arranging this.
