ISPs using nms

A number of ISPs have decided that nms programs are the best CGI programs to offer their customers. This is happening most commonly with formmail. This is almost certainly the result of the recent increases in spam email sent via insecure versions of formmail.

The list of ISPs that we know about is here. If you are would like to be added to this list, please let us know.

  • Verio
  • NetBenefit
  • Pegasus Web Technology
  • IOLS
  • Webhuset Datasenter AS (article in Norwegian)
