nms FormMail - alpha versions

These versions of FormMail are very new and probably full of bugs. They are not yet stable enough for use in production. For now, we strongly recommend that you get your FormMail script from our main script page rather than using these packages.

Files are available in Unix .tar.gz and DOS/Windows .zip formats.

PackageDescription.tar.gz.zip Version
formmail_compat Compatability version, with a standalone FormMail.pl formmail_compat.tar.gz formmail_compat.zip
formmail_modules Modular version, supplied with Perl modules to upload formmail_modules.tar.gz formmail_modules.zip
