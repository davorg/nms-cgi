This is the web site for the nms project. We are a project that supplies free programs for people to use on their web sites. We have no connection at all to do with the people who use our programs. They install and configure the programs themselves. Therefore, if you are here because you have followed a link from a broken web site then we cannot help you. Please see this page for some suggestions on how to contact the people who can help you.

If you need help installing or configuring one of our programs on your site, then please see our support page.

If you have any general enquiries about the nms project, then please contact Dave Cross.
